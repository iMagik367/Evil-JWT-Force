# Testes

O projeto utiliza Pytest para testes automatizados.

## Como rodar os testes
```bash
pytest
```

## Cobertura de testes
```bash
pytest --cov=app tests/
```

## Boas práticas
- Escreva testes unitários e de integração
- Use mocks para dependências externas
- Garanta cobertura mínima de 80%
- Nomeie arquivos de teste como `test_*.py`
