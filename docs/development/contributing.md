# Contribuindo com o Projeto

Contribuições são bem-vindas! Siga as orientações abaixo para colaborar:

## 1. Fork e Clone
- Faça um fork do repositório
- Clone para sua máquina local

## 2. Crie um Branch
- Use nomes descritivos, ex: `feature/novo-modulo`

## 3. Siga o Guia de Estilo
- Use Black, isort, flake8 e mypy
- Execute `pre-commit run --all-files` antes de commitar

## 4. Testes
- Adicione testes para novas funcionalidades
- Certifique-se de que todos os testes passam

## 5. Pull Request
- Descreva claramente sua contribuição
- Relacione issues relevantes

## 6. Código de Conduta
- Seja respeitoso e colaborativo
- Siga as diretrizes do projeto
