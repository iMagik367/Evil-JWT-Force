# Visão Geral da Arquitetura

O sistema é baseado em arquitetura de camadas:
- **Frontend Layer**: Interface web, CLI, dashboards
- **API Gateway Layer**: Rate limiting, autenticação, validação
- **Application Layer**: Lógica de negócio, análise de segurança, IA
- **Data Layer**: Banco de dados, cache, integrações externas

Veja o documento principal de arquitetura para detalhes.
